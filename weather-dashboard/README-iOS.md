✅ **Weather Dashboard para iOS**

## 🎯 Instalação no iPhone/iPad

### Opção 1: Adicionar à Tela Inicial (Recomendado)

**iOS/iPadOS:**
1. Abra `index-ios.html` no **Safari**
2. Clique no botão **Compartilhar** (ícone de seta para cima)
3. Selecione **"Adicionar à Tela Inicial"**
4. Nomeie como "Weather" e clique **Adicionar**
5. Pronto! Agora funciona como um app nativo

### Opção 2: Via URL Direta
- Acesse diretamente: `https://seu-dominio.com/weather-dashboard/index-ios.html`
- Ou abra o arquivo localmente em Safari

## 🚀 Como Usar

### Primeira Vez
1. Instale conforme descrito acima
2. Abra o app
3. Clique em **Configurações**
4. Cole sua chave de API do OpenWeatherMap
5. Salve e aproveite!

### Buscar Cidades
- **Digitar nome**: "São Paulo", "Salvador"
- **Coordenadas**: "-23.55,-46.63" (lat,lon)
- **Localização atual**: Clique no ícone de localização

### Sugestões Aparecem Enquanto Digita
- Selecione a cidade desejada
- Ou pressione Enter para buscar

## 🔧 Obtendo Chave de API

### OpenWeatherMap (Gratuita)

1. Acesse: https://openweathermap.org/api
2. Clique em **Sign Up**
3. Crie uma conta
4. Vá para **API keys**
5. Copie a chave padrão
6. Cole nas configurações do app

**Limites Gratuitos:**
- 1 milhão de requisições/mês
- Até  60 requisições/minuto
- Atualizações a cada 10 minutos

## ⚠️ Troubleshooting

### App não aparece na tela inicial
- Safari → Compartilhar → Adicionar à Tela Inicial
- Ou tente em um iPhone/iPad diferente

### Clima não atualiza
- Verifique internet (Wi-Fi ou dados)
- Recarregue a página
- Verifique a chave de API nas configurações

### Geolocalização não funciona
- Vá em Configurações → Privacidade → Localização
- Ative para Safari ou o App Weather

### Tema escuro não funciona
- Vá em Configurações do app
- Selecione "Automático"
- Ou escolha manualmente

## 📞 Suporte

Para problemas:
1. Limpe cache do Safari (Configurações → Safari → Limpar Histórico)
2. Feche o app completamente
3. Reabra
4. Tente novamente
