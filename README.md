# 🗡️ Terraria Save Editor

Um editor web completo e intuitivo para editar arquivos de save do Terraria diretamente no seu navegador!

## ✨ Características

- 📁 **Carregamento de Saves**: Carregue seus arquivos de save do Terraria (suporte para `.wld` e `.plr`)
- 👤 **Edição de Jogador**: Modifique nome, nível, classe, vida e mana
- 🎒 **Editor de Inventário**: Visualize e edite slots de inventário com suporte a itens
- 📊 **Estatísticas**: Acompanhe deaths, inimigos derrotados, tempo de jogo
- 💀 **Rastreamento de Bosses**: Marque/desmarque bosses derrotados
- 🌍 **Edição de Mundo**: Modifique nome, tamanho, dificuldade e seed
- 💾 **Download de Save**: Baixe seu save editado em JSON
- 📥 **Importar/Exportar**: Trabalhe com arquivos JSON facilmente
- 🎨 **Interface Responsiva**: Funciona perfeitamente em desktop e mobile

## 🚀 Como Usar

### 1. Carregar um Save
- Clique em "Selecionar Arquivo" ou arraste seu arquivo de save para a área destacada
- O editor carregará automaticamente os dados do seu save

### 2. Editar Dados
- **Abas de Navegação**: Use as abas para alternar entre diferentes seções
  - 👤 **Jogador**: Edite dados do personagem
  - 🎒 **Inventário**: Gerencie itens e quantidades
  - 📊 **Estatísticas**: Acompanhe progresso e bosses derrotados
  - 🌍 **Mundo**: Modifique dados do mundo

### 3. Fazer Alterações
- Modifique os valores nos campos de entrada
- Para o inventário, clique em um slot para editar o ID do item
- Marque/desmarque os bosses para rastrear sua progressão

### 4. Salvar Alterações
- Clique em "💾 Salvar Alterações" para baixar o arquivo editado
- Use "📥 Exportar JSON" para exportar em formato JSON
- Use "📤 Importar JSON" para carregar um JSON previamente exportado

## 📋 Seções Disponíveis

### 👤 Jogador
- Nome do jogador
- Nível (0-999)
- Classe (Guerreiro, Arqueiro, Mago, Invocador)
- Vida atual
- Mana atual
- Dificuldade (Normal, Expert, Master)
- Modo Hardcore

### 🎒 Inventário
- 50 slots de inventário
- Visualização de itens com ícones
- Edição rápida de ID e quantidade
- Ferramentas rápidas:
  - Preencher Inventário
  - Limpar Inventário

### 📊 Estatísticas
- Contador de mortes
- Inimigos derrotados
- Bosses derrotados (atualizado automaticamente)
- Tempo de jogo (em minutos)
- Checklist de todos os 18 bosses principais

### 🌍 Mundo
- Nome do mundo
- Tamanho (Pequeno, Médio, Grande)
- Dificuldade (Normal, Expert, Master)
- Tipo de mal (Corrupção/Crimson)
- Seed do mundo

## 📦 Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação
- **CSS3**: Styling responsivo e animações

## 🔒 Privacidade & Segurança

- ✅ Todos os dados são processados **localmente no navegador**
- ✅ Nenhum arquivo é enviado para servidores
- ✅ Seu save permanece **100% privado e seguro**

## 🎨 Design

- Interface moderna e intuitiva

## 📝 Notas Importantes

- Este editor trabalha com a representação JSON dos saves
- Certifique-se de fazer backup de seus saves antes de editar
- Alguns valores extremos podem quebrar o save - use com moderação
- Compatível com todas as versões recentes do Terraria
